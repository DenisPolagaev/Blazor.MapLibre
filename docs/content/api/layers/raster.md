# Raster layer

C# type: `RasterLayer` — use with `RasterTileSource` or raster entries in a style.

Paint: `RasterOpacity`, `RasterHueRotate`, `RasterBrightnessMin`, `RasterBrightnessMax`, `RasterSaturation`, `RasterContrast`, `RasterFadeDuration`, `RasterResampling`.
