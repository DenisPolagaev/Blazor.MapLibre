# Hillshade layer

C# type: `HillShadeLayer` — requires `RasterDEMTileSource`.

Paint: `HillshadeIlluminationDirection`, `HillshadeExaggeration`, `HillshadeShadowColor`, `HillshadeHighlightColor`, `HillshadeAccentColor`, `HillshadeMethod`.

Combine with `SetTerrain` for 3D terrain. See [hillshade example](../../examples/hillshade-terrain.md).
